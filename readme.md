## For Usage and Compilation of Part 1 [Linear Tasks Traversal]:

```$ cd src/linear_tasks
$ make
$ sudo insmod linear_tasks.ko
$ dmesg
$ sudo rmmod linear_tasks
$ dmesg
$ dmesg -c```
